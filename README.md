# NewsReader Starter App

NewsReader
A React-based news application built on a starter template, focused on implementing authentication and access control.
Tech Used

React
Vite
NY Times API
React Context API

Features

Browse NY Times top stories
Search articles by keyword
User authentication (login/logout)
Protected routes requiring authentication
Role-based access control (Regular vs Premium users)
User-specific saved articles
Environment variable API key protection

Setup
npm install
npm run dev
What I Built
Starting from an intentionally insecure starter app, I implemented a full authentication system including login/logout, protected routes, user-specific saved articles, and role-based access between Regular and Premium account types.
